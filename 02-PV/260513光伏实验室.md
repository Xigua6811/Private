
# PID192

“PID192”通常是指**光伏组件抗电位诱导衰减（PID）测试**中的一种常见严苛条件测试，具体对应 **-1000V电压、60℃、85%相对湿度、持续192小时** 的测试序列。

### 核心含义

- **PID（Potential Induced Degradation）**：电位诱导衰减，是光伏组件在高压、高温、高湿环境下，玻璃中的钠离子向电池片迁移，导致组件功率大幅衰减的现象。
    
- **192**：测试持续时间 **192小时**（即连续8天）。
    

### 测试典型条件（PID192）

|参数|典型值|
|---|---|
|测试电压|-1000V 或 -1500V（对应系统电压）|
|环境温度|60℃ ±2℃|
|相对湿度|85% ±5%|
|测试时长|192小时|
|电极连接|组件铝边框接地，电池片接负高压|

### 判定标准

测试前后进行 **最大功率（Pmax）衰减率** 检测：

- **IEC 61215/62804 标准**：衰减 ≤5% 合格
    
- **企业严苛标准**：衰减 ≤2%~3%（如用于双面、高海拔或湿热地区）


# LID60

“LID60”是光伏行业中用于衡量组件**光致衰减（Light Induced Degradation）**能力的一个常见测试或指标表述，具体含义如下：

### 核心定义

**LID60** 是指：在标准测试条件下，对光伏组件进行**累计60 kWh/m²（千瓦时每平方米）的光照暴露**后，测量其最大功率（Pmax）的衰减百分比。

- **LID**：Light Induced Degradation，光致衰减。主要因硼-氧复合体等缺陷导致，在组件首次暴露于光照初期功率快速下降。
    
- **60**：**60 kWh/m²** 的累积辐照量（并非60小时或60%）。
    

### 测试条件与对应时长

- 常用光源：**模拟太阳光（AM1.5，1000 W/m²）** 或 **自然光**。
    
- 对应时长：在 **1000 W/m²** 辐照度下，60 kWh/m² = **60小时** 连续光照。
    
- 实际测试可能先做 **LID5（5 kWh/m²）** 观察快速衰减，再做 **LID60** 评估更长期稳定性。
    

### 常见判定参考（非强制、依赖组件技术）

|组件类型|LID60后典型衰减|业内常见要求|
|---|---|---|
|常规p型多晶/PERC|1.0%~3.0%|<3.0% 合格|
|掺镓p型PERC|0.5%~1.5%|<2.0%|
|n型TOPCon / HJT|**<0.5%**（几乎无LID）|<1.0%|
|早期硼掺杂p型|可能达3%~5%|<5%（IEC 61215要求LID5后<3%）|

> **注意**：IEC 61215:2021 最常用的是 **LID5（5 kWh/m²）要求衰减≤3%**，LID60更多用于企业内部或差异化质保对比。

### 不是“LID 60%”

很多人误以为LID60表示衰减60%，这是完全错误的。60是辐照量单位。即便早期p型多晶LID60也很少超过3%。

### 实际意义

- **区分初始LID（LID5）与持续LID（LID60）**：LID60可以反映组件是否在10-60小时光照后仍持续衰减（可能暗示更多缺陷）。
    
- **对n型组件价值低**：n型几乎无LID，故LID60仅作为基线验证。
    
- **与LETID区别**：LID是光照初期；LETID（高温光致衰减）需额外热注入测试（如75℃+光照）。
    

### 一句话概括

**LID60 = 对光伏组件进行60 kWh/m²的总辐照暴露后，测量其功率衰减比例，用于评估光致衰减的最终稳定水平。**


# LeTID

**LeTID** 是光伏组件中一种比早期 LID 更具破坏性的衰减模式，全称为 **Light and elevated Temperature Induced Degradation**（高温光致衰减）。

与普通的 LID（仅在光照下发生）不同，LeTID 需要在**高温**（通常 >50°C）和**光照**或**电流注入**（载流子注入）的共同作用下才会诱发。它主要困扰 **P型多晶硅 PERC 电池**，但在某些 P 型单晶及先进 N 型电池中也可能出现。

### 1. 核心特征：隐蔽且恢复慢

- **发生时间晚**：通常在组件安装并运行几个月到一年后才明显出现，而不是首几小时。
    
- **衰减幅度大**：严重时功率衰减可达 **10% ~ 15%** 甚至更高，远超普通 LID（~2%）的破坏力。
    
- **缓慢恢复**：在持续光照数年后，功率可能部分恢复，但往往无法回到初始水平。
    

### 2. 根本成因（三个关键因素）

- **氢诱导缺陷**：电池钝化层（如氮化硅）中的**过量氢**，在高温下与电池体内的**金属杂质（主要是铁、铬）** 结合，形成复合中心。
    
- **温度促进**：环境温度必须持续高于 **50°C~80°C**（如沙漠、屋顶、热带地区）。
    
- **载流子注入**：光照或正向电流提供能量，驱动氢-金属复合体的形成。
    

> **通俗理解**：早期 LID 像“着凉”，几分钟就发病；LeTID 像“慢性中毒”，需要高温闷烧几个月才暴发。

### 3. 检测与标准（IEC 63202-2）

测试 LeTID 的工业标准是 **IEC 63202-2**，核心条件为：

|项目|条件|
|---|---|
|温度|**75°C ± 3°C** （加速老化）|
|注入方式|光照 或 正向电流注入（600~1000 mA）|
|持续时间|直至功率稳定或 **648 小时**（27天）|
|合格判定|行业内通常要求 **衰减 < 5%**（部分严苛要求 < 3%）|

**注意**：普通 LID 测试（25°C，60 kWh/m²）无法暴露 LeTID 风险。

### 4. 为什么 LeTID 成为关键问题？

- **PERC 电池的普及**：PERC 生产需要高氢钝化工艺，这恰好是 LeTID 的诱因。多晶 PERC 是重灾区，单晶 PERC 风险稍低。
    
- **高温应用场景增加**：沙漠电站、BIPV（光伏建筑一体化）、屋顶分布式系统常年高温运行。
    
- **长期质保压力**：25 年质保要求首年衰减 <2%，线性衰减 <0.55%/年。若 LeTID 导致 10% 衰减，直接导致质保失效。
    

### 5. 如何降低 LeTID 风险

|方法|原理|效果|
|---|---|---|
|优化烧结/退火工艺|释放掉部分多余氢|显著降低风险|
|更换金属浆料|减少铁、铬等杂质引入|有效|
|使用 N 型硅片（TOPCon/HJT）|N 型几乎无 LeTID 风险|彻底解决|
|组件端“光预处理”|出厂前用强光+高温“预衰减”|加速消耗潜在衰减|

### 6. 常见误区澄清

- **LeTID ≠ 高温 LID**：高温只是触发条件，本质是氢-金属复合，而非硼氧复合。
    
- **LeTID 不是 PID**：PID 是电场驱动钠离子迁移，LeTID 是材料内部化学变化。
    
- **LeTID 仅在多晶出现？**：多晶最严重，但单晶 PERC 也可能出现，只是程度更轻。
    

### 与 LID、PID 的对比总结

|维度|**LID**|**LeTID**|**PID**|
|---|---|---|---|
|触发条件|光照|高温 + 光照/电流|高压 + 高温 + 湿气|
|主要影响对象|P型硼掺杂电池|**P型多晶 PERC**|任何封装不良组件|
|发生时间|首几小时|**几个月后**|数月 ~ 数年|
|衰减幅度|1% ~ 3%|**5% ~ 15%**|5% ~ 30%+|
|可恢复性|不可逆|**缓慢部分恢复**|部分可逆（施加反向电压）|

### 一句话总结

**LeTID 是 P 型 PERC 电池在高温与光照共同作用下，由氢与金属杂质引发的数月的、隐蔽且强烈的功率衰减现象。** 选择 N 型组件或明确提供 LeTID 测试报告（按 IEC 63202-2，75°C/648h，衰减<3%）是规避风险的关键。
