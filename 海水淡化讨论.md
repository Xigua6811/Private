# 合作情况

天津海淡所隶属于海洋局和自然资源部。原先称为一所，中国的海淡项目都有天津海淡所的影子，属于中国海淡的中枢，而且牵头成立了中国海带联盟。天津海淡所和高层领导每半年会有一次互动，而海淡的BD团队至少两周一次会见到。海淡所目前正在执行辽宁营口的预可研，以及潍坊业主工程师的招标工作，是非常重要的战略合作伙伴。

天津海淡所在保税区有1万吨的一个中试基地。

天津海淡所目前与我方已经达成了联合实验室协议，我方共出50万美金，其中由研发走1/3的费用。由项目走其他2/3的费用，海淡所对于要做的研究方向没有特别的诉求，他们只是提出了4个方向，Thomas提出了两个方向，现在正在寻求合作方向的共识。

# R&D方向

向我们建议了三个研究方向，
- 第1个，设备的国产化主要集中在RO膜高压泵以及能量回收装置，
- 第2个，中国海水水质他认为阿夸并不了解中国海水水质，中东的经验不能照搬
- 第3个，数字化和AI，目前在中国项目开发中看不到这些元素。

# 推荐资源

他推荐了沧州的一家民营的海淡工厂，其智能化程度非常高。据说天津海淡所有相应的高压泵以及能量回收装置的相关设备，另外，在高压泵方面，我们与哈电也有相关合作，chairman很希望我们在高压泵和能量回收装置的领域内，把中国的先进设备带到全球。

还推荐了三个资源，在中国具有全流程海带建设经验的有：电建华东院，杭州水处理中心和中交的碧桂园。

另外法液空在闵行的海淡项目，其数字化也做得有可圈可点之处。
