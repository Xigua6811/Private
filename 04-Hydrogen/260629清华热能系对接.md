
## 一、史翊翔团队核心技术体系

团队核心发展思路：**反应高温化、装备撬装模块化、氢基载体（氢/氨/醇）多联产**，区别传统大型现场化工装置。
### 1. 核心赛道1：高温碱性电解槽（双方合作首选方向）
1. 技术参数：常规运行130–150℃，短时耐受180℃；对比常温电解三大优势：
    - 能耗更低、电流密度大幅提升（较常规槽提升3倍），设备体积、耗材大幅缩减；
    - 余热120℃以上高品质热源，可直接耦合海水淡化MED工艺，实现氢水联产，产水量可满足自身180%用水需求，富余淡水对外供给；
    - 负荷调节区间宽（实验室可低至10%额定负荷），适配风光大幅波动工况；
2. 核心自研材料：自研耐高温PTFE隔膜，无分层缺陷，泡点4–5bar，隔氢氧性能优于市面PPS膜，氧中氢含量可控制0.08%，安全性拉低；配套耐高温专用催化剂；
3. 产品路线：高温碱性电解、高温AEM电解（避开PEM贵金属高成本赛道）；
4. 配套系统：自主两相流仿真、全槽模拟、风光储氢一体化群控算法（大唐多伦项目落地验证，采用机理+数据混合建模，可压缩配套储能规模至10%以内）；
5. 成熟度：百千瓦级中试完成，兆瓦级示范在建，计划推进产业化，全套工艺、隔膜、催化剂知识产权归属清华。

### 2. 核心赛道2：3000吨/年撬装式低温低压合成甲醇/合成氨装备

1. 颠覆性创新：摒弃传统大型土建化工厂，工厂预制集装箱模块化撬装设备，单套年产3000吨，多套并联满足大规模产能；
2. 关键突破：中温吸附分离工艺，无需降温液化分离氨，综合成本下降30%、系统灵活度提升一倍；整套装置占地面积极小，公用工程投资降低20%–30%，现场施工大幅简化；
3. 原料兼容：CO、CO₂均可进料，统一转化合成甲醇，适配生物质、工业尾气、绿氢多条路线；
4. 落地资源：鄂尔多斯中美2.3亿国家级风光液态阳光示范项目，拥有充足场地、免费原料气源，可作为中试验证平台；项目由石老师总负责，产业链校友资源完整（东方电气、国电投、浙工大、五环工程等深度合作）。

### 3. 核心赛道3：电解-海水淡化联产一体化系统
依托高温电解高品质余热驱动热法海水淡化，完美匹配ACWA主业，适合中东沿海风光制氢配套，同步解决工业用水、淡水外销收益双重需求。

### 4. 配套辅助技术
1. 高纯氢气变压纯化系统（国内领先，可制备7个9电子级氢气）；
2. 高温燃料电池系统；
3. 风光出力预测、数字孪生管控平台；
4. 小型风光储氢综合集成集装箱（500kW电解+储氢合金+燃料电池一体化，仅4.7㎡占地）。

## 四、双方合作诉求、痛点与合作模式探讨
### （一）ACWA诉求
1. 短期优先落地**高温电解槽合作**：技术成熟、贴近ACWA现有海水淡化主业，易向总部申请研发经费；甲醇撬装项目作为中长期备选；
2. 合作价值：
    - 国内申报项目：绑定清华顶尖技术作为核心创新亮点，补齐外企技术短板；
    - 中东复制：成熟撬装模块化装备适配沙特、阿曼风光基地，解决当地淡水、绿氨/甲醇出口需求；
    - 品牌背书：与清华合作提升企业科创形象，满足政府招商引资技术要求；
3. 自身限制：无制造工厂，仅能做投资、项目开发、市场销售，无法独立生产装备；年度研发预算有限，无力单独承担整套中试装备造价；
4. 市场判断：国内新能源赛道内卷严重、利润微薄，核心盈利市场在中东；中国作为技术孵化基地，完成验证后出海是最优路线。

### （二）清华团队诉求

1. 需求落地中试场景：借助ACWA海外项目渠道、国内政府资源完成装备示范验证；
2. 产业化路径：不涉足终端装备制造，输出工艺包、隔膜、催化剂核心技术，联合装备厂（上海电气、东方电气等央企）量产；
3. 资源置换：ACWA提供国内外项目场景、产业基金、海外市场渠道；清华输出全套专利、研发团队、工艺设计；
4. 风险顾虑：民企合作易出现逆向拆解、知识产权流失，优先央国企、外资投资平台合作。

### （三）多套可行合作方案（现场充分讨论）
#### 方案1：联合中试模式（最优、落地最快）

1. 主体：清华输出工艺、核心隔膜/催化剂、全套设计；ACWA出资搭建兆瓦级高温电解+海水联产示范撬装；引入上海电气等装备厂负责设备加工；
2. 落地场地：鄂尔多斯2.3亿国家级示范项目（气源、场地配套齐全，国拨2000万+企业配套2.1亿资金）；
3. 知识产权：中试成果双方共享，ACWA获得全球商业化授权，清华保留基础研发所有权；
4. 优势：项目有官方背书，可同步申报国家国际合作专项（300–500万经费），双方均无大额设备投入压力。

#### 方案2：工艺包授权+供应商绑定

清华向ACWA输出高温电解、氢水联产成套工艺包，ACWA指定国内装备厂商标准化生产，装备作为ACWA全球项目定点供应设备；清华收取技术授权与持续分成。

#### 方案3：合资初创公司孵化

双方联合产业基金成立合资技术公司：清华以专利技术入股，ACWA中东产业基金现金入股；公司负责技术验证、工艺优化，对接国内制造企业，产品专供ACWA海内外风光氢海水项目；股权可灵活调整，外资基金不并表规避政策限制。

#### 方案4：单纯验证合作

ACWA采购清华核心隔膜、催化剂样品，在自有测试台完成工况验证；验证达标后再推进深度产业化合作，前期投入最低。

### （四）合作顾虑与解决方案

1. 知识产权保密：核心高温隔膜制备工艺清华独立掌控，不对外完整开放，仅配套成套槽体对外合作；
2. 外资合作政策限制：采用“国内技术主体+海外基金参股”架构，合资主体注册国内，规避外资技术管控限制；
3. 制造产能缺口：联动上海电气、东方电气、中能建等央企装备厂完成加工制造，无需双方自建产线；
4. 预算不足问题：依托鄂尔多斯现有项目经费、国家战略性国际合作专项补贴覆盖中试成本，减少ACWA现金投入。

## 五、中东市场独有合作优势（双方共识）
1. 电力成本极低：中东光伏电价全球洼地，风光制氢经济性远超国内；
2. 场地充沛、土地成本低，适合大规模集装箱模块化撬装布局；
3. 海水资源丰富，匹配ACWA淡化主业，氢水联产具备双重收益；
4. 绿氨、绿甲醇海外出口需求旺盛，欧洲、日韩长期稳定采购长单；
5. 沙特2030愿景能源转型政策强力扶持，主权基金大额配套投资；
6. 项目建设周期长，愿意接受中国成熟模块化新技术迭代。

1. 资料输出：石老师向ACWA提供全套高温电解、氢水联产、撬装甲醇装备技术材料、性能数据、专利清单，供吴总内部向CEO、赛总（沙特中方负责人，厦大化工背景，精通中文）汇报、申请研发预算；

3. 高层对接：待内部初步认可后，安排ACWA赛总、氢能全球负责人拜访清华课题组，深度洽谈合资/中试细节；
4. 项目申报：联合申报国家一带一路战略性国际氢能合作专项（300–500万），依托鄂尔多斯示范基地搭建验证平台；
5. 装备厂商对接：同步对接上海电气等央企装备制造方，敲定设备加工配套方案；
6. 中长期：待高温电解验证成熟后，推进3000吨/年撬装合成甲醇装备联合开发，布局中东延布、NEAM大型绿氢化工项目。

## 七、补充次要交流内容
1. 行业现状吐槽：传统大型化工装置土建、管道、阀门投资占比过高（可达项目总投资20%–30%），现场施工周期长、运维复杂，模块化撬装是新能源化工必然趋势；
2. 国内产业痛点：氢能行业现阶段普遍难以盈利，仅依靠政策补贴，出海中东是稳定盈利渠道；
3. 技术路线取舍：PEM电解槽贵金属成本高、降本空间有限，团队放弃深耕，主攻高温碱性、AEM低成本路线；
4. 行业人才与资源：双方拥有大量行业校友、央企、地方政府高层资源，项目申报、落地协调阻力小；
5. 短期障碍：ACWA中国团队人力单薄、研发预算有限，需分步推进，优先小成本中试验证再放大投资。
