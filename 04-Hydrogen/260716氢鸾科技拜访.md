## 氢鸾科技方介绍

柯老师介绍了氢鸾科技的基本情况：

- 创始团队由交大背景人员组成，创业三年，**2025年11月完成首轮融资，宁德时代领投**。
- 全职员工**40人**，另有多名研究生参与研发。位于上海的生产基地约**3,000平方米**，正在建设中。
- 掌握**AEM电解槽全链条核心技术**：膜、催化剂、膜电极、电堆、系统集成均为自制，无外部供应商依赖。
- 产品路线：3.2 kW → 30 kW → 200 kW → 500 kW（兆瓦级验证中）。**30 kW系统已可稳定运行**，200 kW系统已有**二十余套**交付工业现场应用。


## 核心技术与产品

### 3.1 AEM技术特点

- **完全非贵金属化**：阴、阳极均不含贵金属，电流密度可达**15,000 A/m²**，为碱槽的**4–5倍**，相同功率下体积仅为碱槽的**1/5–1/4**。
- **低成本膜**：膜成本预期可降至**100元/㎡以内**，寿命更换成本极低，未来膜可作为易耗品处理。现在PEM膜的成本为10000元/m2。
- **快速启停**：工作温度仅**50–60°C**，冷启动约30分钟，200 kW系统半小时可达稳定工况。
- **压力等级**：**1.6 MPa**，与碱槽相当。
- **运行电耗**：约**4.8 kWh/Nm³**（现场设备数据：0.75%氧中氢），略高于碱槽，但体积和投资优势明显。

### 3.2 已交付业绩

- **南方电网广东肇庆新能源电站**：200 kW系统已稳定运行**3–4个月**，每天约10小时（光伏跟随模式），电解电耗约**4.8 kWh/Nm³**。
- **华能项目**：执行中。
- **验证平台**：30 kW系统目前累计运行约**400–500小时**，数据持续监控中。

### 3.3 波动响应能力

- 已完成光伏波动跟随验证，响应延迟**<1秒**，可在**50%–125%** 负荷范围内快速调节。
- 实验室模拟和现场验证均证明AEM适合直接耦合光伏/风电。


## 合作方向探讨

### 4.1 ACWA方提出的可能方向

Harry Wu提出三点合作思路：

1. **展示**：创新中心需要展示先进技术，体现ACWA在绿氢领域的前瞻布局。
2. **测试/验证**：在ACWA中国创新中心建立测试平台，验证AEM在沙特工况下的可靠性，为大规模推广提供数据支撑。
3. **联合研发**：结合ACWA的应用场景需求与氢鸾的技术能力，开展定向研发。

### 4.2 具体方案讨论

- **小功率系统（3.2 kW）**：氢鸾有现货，约**十几万–二十万元**一套，可直接使用。Harry认为**功率太小，易被质疑为“玩具”** ，不适合向领导展示。
- **30 kW系统**：氢鸾有一台用于展示的现货，可改造后转让，预估**大几十万元**。此前曾以约**145万元**出售类似级别系统。
- **时间节点**：ACWA方希望**2026年9月1日前**到货。氢鸾方表示30 kW系统需要**2–3个月**的组装调试周期（部分零部件需定制采购），如使用现有展示设备改造可大幅缩短时间。
- **租赁方案**：氢鸾表示目前无租赁业务模式。

## 对竞争对手的评价
### 一、对碱水电解（ALK）技术的评价

柯老师和李总对ALK槽的核心判断是：**“走错了方向”**。他们的批评主要集中在以下几个维度：

| 维度             | 评价内容                                                                |
| -------------- | ------------------------------------------------------------------- |
| **大型化困境**      | ALK槽单槽做得越大，一旦出现故障，维修成本极高——“从一拆吊装回来，拆回来运回来，再修好再返回厂，修一次100万””。        |
| **冷启动缺陷**      | ALK槽最怕关停，关停后重新启动的代价极大，夜间只能保温保压维持状态，导致实际运行电耗远高于理论值，经济性被严重削弱。         |
| **密封与可靠性问题**   | 大型ALK槽的一圈垫片换一次就要20万，现场维修极其困难，必须用吊车起吊，送回厂拆解。                         |
| **行业亏损现状**     | 点名三一氢能“估值从30亿跌到12亿”，原因是“前两年的投资者现在想退出来都要亏了”。                         |
| **业主心态变化**     | 通过中能建松原项目经理在公开场合“眼泪都讲掉了”的真实案例，说明ALK槽的实际运行问题已经严重到让业主悔不当初。            |
| **“反向电流”是伪命题** | 很多ALK厂商把寿命短归咎于“反向电流”问题，但氢鸾团队通过验证发现“根本都不是反向电流”，那些走这条路线的厂商“花钱走了一条死路”。 |

**结论**：ALK技术路线在工程实践中面临难以克服的结构性问题，业主已经开始用脚投票。


### 二、对质子交换膜电解（PEM）技术的评价

PEM被定位为**比ALK更昂贵、更不经济的路线**：

| 维度         | 评价内容                                                                                                                                                                                                                                                                                                                                                                                                                          |
| ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **成本不可承受** | “大家搞了这么多年还没把成本降下来”，核心原因是PEM必须用贵金属（铂、铱），材料体系决定成本下限。                                                                                                                                                                                                                                                                                                                                                                            |
| **寿命有限**   | PEM膜的寿命瓶颈尚未突破，系统稳定性和耐久性仍存在根本性风险。                                                                                                                                                                                                                                                                                                                                                                                              |
| **膜很贵**    | - **当前成本**：AEM膜可以做到 **1000元/平米以内**。<br>    <br>- **未来目标**：很有希望降至 **100元/平米**。这意味着一张膜的成本可能只需要几百块钱。他用非常形象的比喻说，AEM膜本质上就是“碳氢体系的东西，跟塑料差不多”，未来完全可以作为消耗品，**“一年换一次又怎么样？”**<br>    <br>- **电堆成本**：基于如此低廉的膜成本，他估算一个电堆的膜材料成本可能仅需 **2000元左右**。<br>    <br><br>- PEM膜：因贵金属而价格高昂<br><br>柯老师将PEM膜的高成本归咎于其必须使用贵金属的材料体系，他通过一个生动的比喻来说明：<br><br>同样是1平方米，PEM膜有36道工序，如果每道工序0.8%的成品率，损失是 **0.8%的36次方**；而PEM膜因为依赖铱等贵金属，结果就是“完蛋了”，成本极其高昂。而AEM膜只有三道工序。 |


### 三、对竞争对手的定位与评价

柯老师和李总将竞争对手分为三类：

| 类型 | 代表性企业 | 评价 |
|------|------------|------|
| **ALK头部企业** | 某央企背景企业、某已上市公司 | 前者因ALK槽陷入困境，“很难受”；后者因ALK业务拖累，“估值从30亿跌到12亿”，“三一可能退出减仓”。 |
| **PEM头部企业** | 某老牌PEM企业、某被收购企业 | 前者“市值从700亿跌到100亿”，后者“被央企收购”。 |
| **AEM同行** | 国内多家AEM创业公司 | 大部分AEM企业仍处于“刚起步阶段”，还在用**贵金属催化剂**、用**实验室条件**做迭代。没有解决工程化问题，没有在**真实的工业场景**下做验证。氢鸾是唯一“真正在工业现场跑过、用数据迭代产品”的AEM企业。 |


### 四、对AEM行业现状的判断

柯老师和李总认为AEM行业整体处于“早期探索阶段”，大部分同行仍在以下方面挣扎：

1. **催化剂路线落后**：不少同行还在用**贵金属**，而氢鸾已经实现**完全非贵金属化**。
2. **供应链依赖外部**：大部分AEM企业依赖外部供应商提供膜、催化剂等核心材料，无法形成工程闭环。
3. **缺乏真实场景验证**：“很多做AEM的连现场都没去过”，只在实验室条件下做论文式的优化，无法解决真实工业环境下的运行问题。


### 五、氢鸾的差异化定位

氢鸾被定位为**“正向做产品”而非“讲故事”的公司**：

- **全链条自研**：催化剂→膜→膜电极→电堆→系统集成，全部自研自制，不与不可靠的供应商合作。
- **工程化导向**：团队大部分人员在一线，“跑在真实的工业场景”中收集数据、迭代产品。
- **有实际交付业绩**：已交付20余套200 kW工业级系统，有南网、华能等央企客户背书。
- **非贵金属化**：阴、阳极均不含贵金属，成本结构接近ALK，性能密度接近PEM，定位为**“ALK的成本 + PEM的性能”**。


### 六、总结

柯老师和李总的整体叙述逻辑是：

1. **ALK走错了方向**——大型化带来的经济性和可靠性问题已经暴露，行业陷入亏损。
2. **PEM走不通**——贵金属成本降不下来，市场信心崩溃。
3. **AEM有同行，但还没人做到位**——大部分停留在实验室阶段，用贵金属，靠供应商供料，缺乏工业场景验证。
4. **氢鸾是第一个真正跑通的AEM企业**——全链条自研、非贵金属化、工业现场验证、有业绩、有宁德时代背书。
