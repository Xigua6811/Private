
# 反渗透海水淡化预处理技术：原理、工艺与工程应用系统性综述

## 一、预处理的基本概念与核心使命

### 1.1 何为海水淡化预处理？

海水淡化预处理是指在一级处理（反渗透脱盐）之前，通过物理、化学或生物手段，对原海水进行净化处理的一系列工艺单元的总称。其目标并非脱盐，而是**为后续反渗透（RO）膜系统提供满足进水水质要求的、稳定可靠的给水**。

### 1.2 海水中的污染物分类及其危害（莫晓媛等，2023；尹新星等，2025）

海水中对RO系统有害的物质，按其存在形态可分为两大类：

| 形态类别 | 定义（以0.45μm滤膜为界） | 典型物质 | 主要危害 |
| :--- | :--- | :--- | :--- |
| **颗粒态** | 被0.45μm滤膜截留的物质 | 悬浮泥沙、粘土、藻类、细菌、原生动物、后生动物、不溶性有机物 | 堵塞膜流道，增加跨膜压差；磨损膜表面 |
| **溶解态** | 能透过0.45μm滤膜的物质 | 无机盐离子（Ca²⁺、Mg²⁺、SO₄²⁻等）、溶解性有机物、胶体（铁/铝/硅的水合物）、溶解气体（O₂、CO₂、H₂S） | 胶体污染；结垢（微溶盐析出）；有机污染；生物污堵 |

### 1.3 预处理的四大核心目标（综合多篇文献）

1.  **去除悬浮固体与胶体**：降低浊度和SDI（淤泥密度指数），防止颗粒物在膜表面沉积。
2.  **控制微生物与生物污堵**：杀灭或去除细菌、藻类，防止其在膜表面形成生物膜（生物污堵是最难清除的污染类型，占膜污染案例的45%以上）。
3.  **抑制结垢**：通过调节pH或去除部分硬度离子，防止CaCO₃、CaSO₄、BaSO₄、SrSO₄等微溶盐在膜浓水侧超过溶度积而结晶析出。
4.  **消除氧化性物质**：去除海水中的余氯或其他氧化剂，保护RO聚酰胺复合膜不被氧化降解。

### 1.4 反渗透膜进水水质通用指标（蔡晶，2020；张岩岗等，2018）

| 指标 | 物理/化学含义 | 控制目标 | 最佳值 |
| :--- | :--- | :--- | :--- |
| **SDI₁₅（淤泥密度指数）** | 在2.1 bar压力下，0.45μm微孔滤膜在15分钟内因胶体/颗粒堵塞造成的流量衰减速率。反映水中胶体及微细颗粒的污染潜势。 | ≤5 | <3 |
| **浊度（NTU）** | 利用光散射原理测量的水中悬浮物对光线的阻碍程度。表征肉眼可见的悬浮泥沙、不溶物的含量。 | <1.0 NTU | <0.2 NTU |
| **余氯（Cl₂）** | 水中残留的游离氯或化合氯。强氧化剂，会攻击RO膜的交联结构。 | <0.1 mg/L | — |
| **水温** | 海水的物理温度。影响水的粘度和膜通量。 | 5~45℃ | 20~30℃ |

> **重要数据**：徐秀萍等对RO海水淡化性能衰减的案例分析显示，**因预处理效果不佳导致膜污染引起的性能衰减约占总因素的77%**，膜元件机械损坏仅为次要因素（张岩岗等，2018）。预处理的重要性由此可见。

### 1.5 预处理工艺研究热度与实际应用格局（蔡晶，2020）

从过去十年的研究文献统计来看：
- 超滤（UF）预处理研究热度最高，占29%
- 微滤（MF）占17%
- 混凝沉淀占18%

然而，在实际大型工程应用中，由于膜法预处理投资成本过高，**传统预处理技术（混凝沉淀、介质过滤、气浮）仍占绝大部分份额**。研究热度与实际应用之间存在显著落差。


## 二、预处理核心工艺技术原理与设计要点

### 2.1 混凝与絮凝（化学预处理的核心单元）

#### 2.1.1 基本概念（莫晓媛等，2023；高天鸿等，2024）

- **混凝（Coagulation）**：指投加无机盐类混凝剂（如三氯化铁FeCl₃、聚合氯化铝PAC）后，药剂水解产生的高价阳离子**压缩水中胶体颗粒表面的双电层**，降低Zeta电位，消除颗粒间的静电排斥力，使原本稳定分散的微小颗粒脱稳凝聚成微小的“矾花”。这一过程的核心机制是**电荷中和**。
- **絮凝（Flocculation）**：指在混凝之后，通过机械搅拌或水力作用，或投加高分子助凝剂（如聚丙烯酰胺PAM），利用高分子的**吸附架桥**和**网捕卷扫**作用，将微小矾花聚合成体积更大、密度更高的粗大絮凝体。这一过程的核心机制是**碰撞成长**。

#### 2.1.2 常用药剂
- 混凝剂：三氯化铁、聚合氯化铝（PAC）、硫酸铝等
- 助凝剂：聚丙烯酰胺（PAM）等

#### 2.1.3 絮凝池设计要点（李娴，2025；莫晓媛等，2023）
絮凝效果受原水水质、pH值、水温、混合方式等因素影响。以广泛应用的**网格絮凝池（翼片隔板反应池）**为例，其设计流速分三级：

| 级别 | 水流流速 | 功能 |
| :--- | :--- | :--- |
| 第一级 | ~0.12~0.13 m/s | 高强度碰撞，使微小颗粒脱稳 |
| 第二级 | ~0.08~0.09 m/s | 中等强度，絮体成长 |
| 第三级 | ~0.05~0.06 m/s | 缓和搅拌，絮体长大密实 |

总停留时间一般为**15~20分钟**。对低温低浊海水，可适当延长絮凝时间或采用污泥回流工艺。

### 2.2 沉淀（重力固液分离）

#### 2.2.1 基本原理（斯托克斯定律）（莫晓媛等，2023）
沉淀是利用水中悬浮颗粒与水的密度差，使颗粒在重力作用下下沉。根据斯托克斯（Stokes）公式，**颗粒沉降速度与其粒径平方成正比，与水的粘度成反比**。因此，颗粒越大、水温越高（粘度低），沉淀效果越好。

#### 2.2.2 浅池理论（斜板/斜管沉淀池的设计依据）
该理论指出：沉淀池的处理效率（即表面负荷）与沉淀面积成正比，而与池深无关。通过设置密集的斜板或斜管，将沉淀池分隔成众多浅层沉淀单元，**大幅缩短颗粒沉降距离**（从数米缩短至数厘米），从而成倍提高单位占地面积的处理能力。

#### 2.2.3 主要池型对比（蔡晶，2020；莫晓媛等，2023）

| 参数 | 平流式沉淀池 | 斜管（板）沉淀池 | 机械加速澄清池 | 高密度沉淀池 |
| :--- | :--- | :--- | :--- | :--- |
| 上升流速 | — | — | 0.7~1.1 mm/s | 3.3~3.5 mm/s |
| 回收率 | ~90% | ~90% | ~95% | ≥95% |
| 占地 | 大 | 小 | 较小 | 节省约20% |
| 排泥方式 | 机械排泥 | 机械/重力 | 机械排泥 | 机械排泥 |
| 药耗 | 较多 | 较多 | 较少 | 较少 |
| 优点 | 结构简单，对水质水量适应性强 | 沉淀效率高，占地小 | 处理效果好 | 集成度高，絮体密实，沉降极快 |
| 缺点 | 占地面积大 | 斜管为耗材，需定期更换 | 动设备多，运维复杂 | 设备成本高约20% |

> **高密度沉淀池的技术亮点**（蔡晶，2020）：通过**污泥外循环回流**，使搅拌反应池保持较高的固体浓度，利用回流污泥的接触絮凝作用，使新生絮体以回流污泥颗粒为“核”加速成长，形成的絮体更加密实、沉重，沉降速度极快（上升流速达3.3~3.5 mm/s），从而显著节省占地和药耗。

### 2.3 气浮（针对轻质颗粒的固液分离）

#### 2.3.1 基本原理（亨利定律应用）（莫晓媛等，2023；蔡晶，2020）
气浮通常采用**加压溶气气浮（DAF）**，其原理基于亨利定律：气体在液体中的溶解度与压力成正比。DAF的工艺流程为：
1.  **加压溶气**：在加压条件下（通常0.3~0.5 MPa），将大量空气强制溶解于水中，形成过饱和溶气水。
2.  **减压释放**：将溶气水骤然减压至常压释放，水中溶解的空气以**直径20~30μm的微细气泡**形式析出。
3.  **黏附上浮**：微气泡与水中絮体发生物理黏附。当气泡-絮体复合体的总体密度小于水时，便在浮力作用下快速上浮至水面。
4.  **刮渣去除**：水面形成的浮渣层由刮渣机刮除。

#### 2.3.2 DAF的独特优势与适用场景（多篇共识）
- **对藻类的去除**：DAF可去除90%~99%的藻类细胞，而混凝沉淀法仅能去除60%~90%。这是因为藻类细胞比重接近水、沉降极慢，而DAF通过气泡主动黏附，能高效捕捉。
- **对油脂的去除**：DAF对油污去除率约80%，远优于沉淀。
- **低温低浊水处理**：低温时水粘度大，颗粒沉降更慢，而气浮受温度影响相对较小。
- **占地小、停留时间短**：DAF表面负荷可达10~40 m³/(m²·h)，停留时间仅10~20分钟，约为沉淀池的1/10。

#### 2.3.3 局限性
配套设备多（溶气罐、释气器、空压机、刮泥机），电耗和后期管理成本较高；对进水与溶气水的均匀混合要求严格。

### 2.4 介质过滤（深层精细过滤）

#### 2.4.1 过滤机理（并非简单筛分）（莫晓媛等，2023）
介质过滤（如石英砂、无烟煤滤池）的截留机理并不仅靠滤料孔隙的“筛网”作用，而是包括：
1.  **惯性碰撞**：颗粒随水流运动时因惯性撞上滤料表面而被截留。
2.  **扩散吸附**：微小颗粒（<1μm）受布朗运动影响，扩散至滤料表面被吸附。
3.  **黏附作用**：滤料表面形成的生物黏膜及物理化学吸附力，将微絮体“粘”在滤料表面。
4.  **沉淀作用**：颗粒在滤层孔隙中因重力自然沉降。

因此，介质过滤是一种**深层过滤**，滤层厚度直接影响截污容量和出水水质。

#### 2.4.2 重力式过滤 vs 压力式过滤（蔡晶，2020；莫晓媛等，2023）

| 项目 | 重力式过滤（以V型滤池为代表） | 压力式过滤（多介质过滤器/卧式砂滤） |
| :--- | :--- | :--- |
| 驱动方式 | 重力水头（约1.5~2.5m） | 水泵增压（0.15~0.30 MPa） |
| 过滤速度 | 6~12 m/h | 8~20 m/h |
| 出水浊度 | <2 NTU（进水<10 NTU时） | <0.5 NTU（进水<5 NTU时） |
| 滤料特点 | 均粒径石英砂（0.9~1.2 mm），滤层厚0.9~1.5m | 多层滤料（无烟煤+石英砂+磁铁矿），粒径较小 |
| 抗水质波动 | 强（滤层厚、停留时间长） | 弱（滤层薄、停留时间短） |
| 藻华适应性 | 较好（重力式砂滤是更合适的选择） | 差（高压下藻细胞破裂，释放有机物加剧RO污染） |
| 反洗方式 | 气+水联合反洗+表面扫洗 | 水反洗（必要时加气） |
| 占地 | 大 | 小 |
| 投资 | 土建高，设备低 | 设备高，土建低 |

#### 2.4.3 典型三层滤料组成（蔡晶，2020）

| 层次 | 滤料 | 粒径（mm） | 滤层厚度（mm） | 功能 |
| :--- | :--- | :--- | :--- | :--- |
| 上层 | 无烟煤 | 0.8~1.6 | 450 | 截留大颗粒，轻质，纳污量大 |
| 中层 | 石英砂 | 0.5~0.8 | 230 | 主过滤层，去除中等颗粒 |
| 下层 | 磁铁矿砂 | 0.25~0.5 | 70 | 精细截留，支撑层 |

这种**上粗下细**的级配方式，使整个滤层都能发挥截留作用，避免表面堵塞，延长过滤周期。

### 2.5 膜法预处理（MF/UF）

#### 2.5.1 核心分离机理（绝对物理筛分）（高天鸿等，2024；张岩岗等，2018）
与介质过滤的深层吸附不同，MF（微滤，孔径0.05~5μm）和UF（超滤，孔径0.01~0.1μm）膜的分离依靠**膜表面均匀的微孔进行严格的物理尺寸排阻**：大于膜孔径的物质被绝对截留在膜表面，小于孔径的水和溶解性物质则透过膜。这是一种**绝对屏障**，产水浊度极低且稳定（一般<0.2 NTU，SDI<3）。

#### 2.5.2 中空纤维膜的结构与过滤方式

**（1）内压式（由内向外过滤）**
- 原水从中空纤维膜丝的内腔流过，透过膜壁向外收集产水。
- **优点**：可通过高速内冲洗将内腔污染物排出，清洗效果好。
- **缺点**：内腔直径仅约1mm，流道狭窄，对原水颗粒物含量要求高（浊度<5 NTU），易堵塞，抗冲击性差。

**（2）外压式（由外向内过滤）**
- 原水在膜丝外侧流动，透过膜壁进入内腔收集产水。
- **优点**：外侧流道宽阔，可处理高浊度原水（外压式可达15 NTU）；污染物附着在膜外表面，反洗时易冲落；可采用**气/水双洗**（空气擦洗+水反冲），通量恢复效率高；抗冲击性强。
- **缺点**：壳体存在清洗死角。

#### 2.5.3 运行模式：全流（死端）过滤 vs 错流过滤（高天鸿等，2024；张岩岗等，2018）

| 项目 | 全流过滤（死端过滤） | 错流过滤 |
| :--- | :--- | :--- |
| 水流方向 | 原水全部垂直透过膜面，无浓水排放 | 原水与膜面平行流动，部分透过膜成为产水，部分携带截留物以浓水形式排出 |
| 能耗 | 低 | 高（需保持流速） |
| 回收率 | >95% | 90%~95% |
| 膜污染速率 | 快（截留物全部堆积在膜面） | 慢（剪切力抑制浓差极化并带走污染物） |
| 适用原水 | 浊度<5 NTU，水质稳定 | 浊度>10 NTU，悬浮物含量高 |

#### 2.5.4 超滤/微滤膜材料对比（蔡晶，2020；高天鸿等，2024）

| 膜材质 | 优点 | 缺点 |
| :--- | :--- | :--- |
| **PVDF（聚偏氟乙烯）** | 耐化学腐蚀强、抗污染、机械强度大、伸长率高 | 成本相对较高 |
| **PES（聚醚砜）** | 机械强度较大、易成型、生物融合性好、耐热 | 耐碱性略逊于PVDF |
| **陶瓷膜（Al₂O₃/SiO₂等）** | 化学稳定性极好、机械强度大、抗微生物、耐高温、通量极高（400~500 LMH）、寿命长达10年 | 初始成本高、不耐热冲击 |

#### 2.5.5 陶瓷膜的技术特点（蔡晶，2020）
陶瓷膜是**亲水性与坚固性的完美结合**，具有极佳的耐酸碱能力（pH使用范围1~13，远超有机膜的2~11），可承受热氢氧化物或酸的清洗。其通量是有机超滤膜的4~5倍，使用寿命可达10年（有机膜仅3~5年）。然而，**成本通常比高分子膜高**，且不能长时间经受温度急变或较大的热冲击场合。

### 2.6 连续膜过滤（CMF）技术（高天鸿等，2024）

连续膜过滤是以中空纤维超/微滤膜组件为核心，配备管道、阀门、自清洗模块、加药模块和自动控制系统，形成的一个可连续自动化操作的闭环系统。

**典型工艺流程**：
过滤 → 气/水双洗（压缩空气使膜丝摇摆摩擦，剥落污染物）→ 水反冲洗 → 自然排污 → 化学增强反洗（低浓度药剂）→ 恢复性化学清洗（高浓度药剂循环浸泡）

**CMF技术特点**：
- 采用高强度聚丙烯（PP）中空纤维膜，抗氧化、耐污染
- 利用PP膜的弹性，微孔可扩张，结合压缩气体反洗，膜通量恢复率高
- 模块化设计，占地小，自动化水平高
- 可超低压运行（0.1 MPa以下），节能环保

**影响CMF系统的关键因素**：进水水质、水温（随温度降低，水粘度增加，膜通量下降）、膜通量选择、跨膜压差。


## 三、预处理工艺选择的量化决策框架

### 3.1 预处理工艺选择的影响因素（李娴，2025；莫晓媛等，2023）

1.  **水质因素**：海水中杂质的类型（颗粒/胶体/溶解物）和浓度（浊度、TOC、藻类计数、油脂含量等）。
2.  **进水水质要求**：后续RO系统对SDI、浊度的严格要求决定预处理深度。
3.  **经济因素**：初期投资、运行能耗、药剂费用、膜更换费用、维护管理成本。
4.  **环境与场地**：水温、季节变化（藻华期）、可用地面积、取水方式（深海管道/开放式进水口）。
5.  **突发事件风险**：赤潮、藻华、油泄漏、化学品泄漏等。

### 3.2 基于水质指标的量化工艺推荐（尹新星等，2025）

尹新星等（2025）收集了全球大型膜法海水淡化厂的预处理工艺案例，提出了基于**浊度、SDI₁₅、TOC**三项核心指标的量化决策表：

| 水源水质条件 | 推荐预处理工艺 | 技术逻辑 |
| :--- | :--- | :--- |
| 浊度<0.1 NTU，SDI<2，TOC<1 mg/L | **仅设RO保安过滤器** | 深海取水，水质已极清，无需物化预处理 |
| 浊度<5 NTU，SDI<5，TOC<1 mg/L | **单级过滤（双介质过滤器/MF/UF）+保安过滤器** | 选用UF/MF时可不加混凝剂 |
| 浊度<30 NTU，SDI>5，TOC<4 mg/L（中度藻华） | **单级双介质过滤+混凝剂+保安过滤器** | 需投加混凝剂使细小颗粒脱稳，便于介质过滤 |
| 浊度<50 NTU，SDI>5，TOC≥4 mg/L（严重藻华或油污染） | **沉淀（DAF或混凝沉淀）+单级双介质过滤+保安过滤器** | 先大幅削减悬浮负荷，再精细过滤 |
| 浊度>50 NTU，SDI>5，TOC≥4 mg/L（严重藻华或油污染） | **高速沉淀+两级双介质过滤+保安过滤器** | 两级砂滤串联（粗滤→精滤）确保SDI达标 |

**关于沉淀工艺的选择原则**（尹新星等，2025）：

- **选DAF**：藻华发生频繁、对藻类去除率要求高、水源有油污染风险、占地紧张
- **选混凝沉淀**：原水浊度高、出水浊度要求不高、场地不受限

### 3.3 常规海滨预处理工艺对比（张岩岗等，2018）

| 工艺 | 原理 | 出水水质 | 运行管理 | 运行费用 | 抗冲击负荷 | 药剂使用 |
| :--- | :--- | :--- | :--- | :--- | :--- | :--- |
| ACTIFLO高效混凝沉淀池 | 斜板+微砂沉淀 | SS<20 mg/L | 简单 | 低 | 好 | 铁盐，可不加助凝剂 |
| 高密度澄清池 | 澄清+污泥回流 | SS<20 mg/L | 简单 | 较高 | 好 | 铁盐+PAM，投加量大 |
| 气浮-沉淀池 | 气浮澄清+下层过滤 | SS<10 mg/L | 简单 | 低 | 好 | 铁盐或铝盐，可不加助凝剂 |
| 气浮-滤池 | 上层气浮+下层过滤 | SS<5 mg/L | 复杂 | 较高 | 较差 | 铁盐+PAM，加药量难控制 |

## 四、砂滤法 vs 膜滤法（UF/MF）的深度博弈

这是大型海水淡化工程中最核心的预处理工艺路线之争（张岩岗等，2018；尹新星等，2025）：

| 对比维度 | **砂滤法（传统介质过滤）** | **膜滤法（UF/MF）** |
| :--- | :--- | :--- |
| **颗粒去除机理** | **深层黏附过滤**：依靠滤料表面生物膜及范德华力吸附，非绝对屏障，出水受水温、流速影响波动 | **绝对物理筛分**：依靠特定微孔直接截留，出水浊度极低（<0.2 NTU）且极其稳定 |
| **出水浊度/SDI** | 浊度<1 NTU，SDI<5 | 浊度<0.2 NTU，SDI<3 |
| **可溶性有机物去除** | **具备生物过滤功能**：滤层中生物膜可降解原水中20%~40%的可溶性有机物 | **几乎无效**：停留时间极短，无生化作用；仅截留大分子量有机物 |
| **藻华适应性** | **重力式砂滤更合适**：藻类可在滤床深处被生物膜黏附降解 | **不适用**：高压或外压进水使藻细胞破裂，释放胞内有机物（IOM），IOM穿透膜后**严重加剧RO生物污堵** |
| **突发事件适应性** | **强**：油污/化学品泄漏后可通过换砂或强力反洗恢复 | **弱**：油污或有机溶剂会造成膜不可逆损坏，需整批更换 |
| **系统水回收率** | ~90%（反洗耗水量大） | ≥95%（反洗水可直接回流） |
| **能耗** | 重力式约0.05 kWh/m³；压力式与膜相当 | 0.2~0.4 kWh/m³；低温时通量下降明显 |
| **环保压力** | 反洗多为物理冲洗，极少加药剂，废水易处理 | 需频繁化学增强反洗（CEB）和化学清洗（CIP），产生大量难处理含酸/碱/氧化剂废水 |
| **投资与运行费用** | **较低** | **高**（膜组件昂贵，寿命3~7年） |
| **占地** | 大 | 小20%~50% |

### 选择建议（尹新星等，2025；张岩岗等，2018）：

- **优先选择砂滤法**：海水水质波动大、可溶有机物多、藻华频发、场地不受限、投资方注重投资和运行费用、对环境污染敏感。
- **优先选择膜滤法**：海水水质较好、水源突发事件少、占地紧张、希望减少RO系统维护管理工作量。

## 六、总结与行业趋势

### 6.1 核心共识（综合六篇文献）

1.  **预处理是RO系统成败的关键**：约77%的RO膜性能衰减与预处理不当有关。合理预处理可提高系统运行效率20%~30%。
2.  **组合工艺是主流**：“混凝/气浮 + 介质过滤（砂滤）/超滤”是最常见的配置，极少采用单一工艺。
3.  **没有万能工艺，只有最适工艺**：必须基于**长期连续的海域水质监测**（特别关注藻华期、台风期浊度峰值、油污风险），结合**中试试验**确定最佳工艺参数。
4.  **生物污堵是最棘手的问题**：微生物（细菌、藻类）形成的生物膜最难清除，且残留生物膜会加速二次污染。DAF在应对藻类和油脂方面的独特价值正被重新高度认可。
5.  **超滤并非万能的升级替代品**：尽管产水水质极高，但对原水水质变化具有**脆弱性**（藻华破裂、油污损伤、化学品泄漏），且化学药剂依赖度高、投资运维费用高。在大型市政海淡项目中并未完全取代传统砂滤。
6.  **重力式砂滤在大型工程中仍占主导**：尤其是在水质波动大、藻华频发的海域，其抗冲击能力、生物过滤功能、突发事件耐受性和经济性综合优势显著。

### 6.2 技术与政策趋势

1.  **一体化与集约化**：趋向于将絮凝、沉淀/气浮、过滤集成于同一构筑物（如高密度沉淀池、Acti-DAFF一体化气浮滤池），减少占地和投资。
2.  **智能化与自动化**：通过PLC自动监控进出水浊度、压差，实现全自动按需反洗与精准加药，减少人为操作波动。
3.  **国产化进程加速**：国产超滤膜性能提升、成本降低，国产化率不断提高，推动膜法预处理在更多项目中应用。
4.  **政策强力推动**：中国“十四五”规划明确提出“推进海水淡化和海洋能规模化利用”，2022年《扩大内需战略规划纲要》要求在沿海缺水城市建设大型海水淡化设施，膜法（RO）占比超60%，预处理市场需求将持续增长。
5.  **“砂滤+膜滤”可能走向互补融合**：越来越多工程采用“砂滤+UF/MF”的双重保障配置（如河北丰丰海基），利用砂滤承担主要悬浮物负荷、UF/MF作为绝对安全屏障，兼顾经济性与可靠性。
